### 前端工程化

#### 1.工程化

- 模块化（js的模块化、css的模块化、资源的模块化）
- 组件化（复用现有的UI结构、样式、行为)
- 规范化（目录结构的划分、编码规范化、接口规范化、文档规范化、Git分支管理)
- 自动化（自动化构建、自动部署、自动化测试)

#### 2.什么是前端工程化

- 前端工程化指的是:在企业级的前端项目开发中，把前端开发所需的工具、技术、流程、经验等进行规范化、标准化。
- 好处：前端开发自成体系，有一套标准的开发方案和流程

#### 3.前端工程化的解决方案

- 早期的前端工程化解决方案
  - grunt
  - gulp
- 目前主流的前端工程化解决方案
  - webpack
  - parcel
